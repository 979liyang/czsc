# 第一类买卖点绘制验证

**Purpose**: 确认 K 线图上一买、一卖标记的绘制符合规格（Phase 16/17）。  
**Feature**: [tasks.md](../tasks.md) Phase 18 / T032

## 验收要点

- [ ] **一买锚点**：锚点价格与该根 K 线的 low 一致（或与分析 API 返回的该事件 price 一致）。
- [ ] **一买连接线**：从锚点向下延伸，垂直长度为锚点价格的 **3%**；竖线为**虚线**；无横向射线。
- [ ] **一买文案**：文案「一买」位于连接线下方端点处。
- [ ] **一卖锚点**：锚点价格与该根 K 线的 high 一致（或与分析 API 返回的该事件 price 一致）。
- [ ] **一卖连接线**：从锚点向上延伸，垂直长度为 **3%**；竖线为**虚线**；无横向射线。
- [ ] **一卖文案**：文案「一卖」位于连接线上方端点处。
- [ ] **与表格一致**：图表上一买/一卖的时间与麒麟分析页「第一类买点」「第一类卖点」表格中的 `dt` 对应一致。

## 如何验证

1. 启动后端与前端，打开 K 线图页（如 `/chart` 或对应路由）。
2. 打开「分析买卖点」开关，确保「一买」「一卖」子开关为开启。
3. 选择有第一类买卖点数据的标的与周期（可先在麒麟分析页执行一次分析确认有 buy1_events / sell1_events）。
4. 对照上述清单逐项检查；若有偏差则按 tasks.md Phase 12/16/17 排查绘制逻辑与数据源。
